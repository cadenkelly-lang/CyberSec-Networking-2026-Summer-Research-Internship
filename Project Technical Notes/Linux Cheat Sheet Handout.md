| Command                   | What It Does                                                                     |
| ------------------------- | -------------------------------------------------------------------------------- |
| `pwd`                     | Print working directory that shows where you are                                 |
| `cd`                      | Change directory (`cd /path` = absolute, `cd path` = relative, no leading slash) |
| `ls`                      | List files in the current directory                                              |
| `find`                    | Search for files by name, type, or location                                      |
| `grep`                    | Search text inside files for a pattern                                           |
| `\|` (pipe)               | Sends output of one command into the next command                                |
| `cat`                     | Print entire file contents to the screen                                         |
| `nano`                    | Text editor for creating/editing files                                           |
| `touch`                   | Create an empty file                                                             |
| `sudo`                    | Run a command with admin/root privileges                                         |
| `ip a`                    | Show network interfaces and IP addresses                                         |
| `ping`                    | Test connectivity to another host                                                |
| `ip route`                | Show the routing table                                                           |
| `traceroute`              | Show the path (hops) packets take to reach a destination                         |
| `nslookup`                | Look up the IP address behind a domain name (DNS lookup)                         |
| `ssh user@host`           | Log into a remote computer                                                       |
| `scp file user@host:path` | Copy a file from your machine to theirs                                          |
